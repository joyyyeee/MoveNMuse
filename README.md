# MoveNMuse
github: https://github.com/marinankato/MoveNMuse.git

Move N Muse is a full-stack web platform for discovering, booking, and managing dance and music classes. It includes session scheduling, room hire management, and secure online bookings â€” all within a clean, responsive interface designed to make studio management simple for small businesses.

----
## Tech Stack

**Frontend:**
- React (Vite)
- Tailwind CSS
- Redux Toolkit
- React Router

**Backend:**
- Node.js (v22)
- Express.js
- MongoDB (Mongoose)
- JWT Authentication

**DevOps & Hosting:**
- Azure App Service (Linux)
- Azure Pipelines (CI/CD)
- GitHub for source control

----
## Project Structure

The repo is divided into two main folders:
/server for backend code and /client for frontend code.
